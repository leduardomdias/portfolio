Projetos Recentes

Fortune Cookie

Um simples projeto em React onde o usuário pode abrir um biscoito da sorte e receber uma mensagem aleatória.


StopWatch

Este é um aplicativo de cronômetro desenvolvido em React. Ele permite iniciar, pausar e resetar a contagem do tempo com um design simples e intuitivo.

To-Do List Application

Um aplicativo de lista de tarefas elegante e intuitivo, desenvolvido com JavaScript, HTML e CSS. Ele permite que os usuários adicionem, editem, excluam e marquem tarefas como concluídas. Os dados das tarefas são armazenados no armazenamento local do navegador, garantindo que permaneçam salvos mesmo após atualizar a página ou fechar o navegador.

Currency Converter

Este é um aplicativo simples de Conversor de Moeda desenvolvido usando HTML, CSS e JavaScript. Ele permite que os usuários convertam valores de uma moeda para outra com base em taxas de câmbio fixas.
